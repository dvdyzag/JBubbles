JBubbles
========
Puzzle Bobble game made in Java

Compile
```bash
mkdir bin
cd src
javac gt/dvdyzag/JBubbles.java -d ../bin/
cp -r imagenes/ ../bin
```
Run
```bash
cd ../bin/
java gt.dvdyzag.JBubbles
```
Make Jar
```bash
jar cfe JBubbles.jar gt.dvdyzag.JBubbles acercade gt imagenes
```
Run Jar
```bash
java -jar JBubbles.jar

```
